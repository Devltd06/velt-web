# App Screenshots

Add your app screenshots here with the following names:

1. `home.png` - Home feed screen
2. `profile.png` - User profile screen
3. `explore.png` - Explore/discover screen
4. `billboards.png` - Billboards screen
5. `marketplace.png` - Marketplace screen

## Recommended Specifications

- **Dimensions:** 1170 x 2532 pixels (iPhone 14 Pro resolution)
- **Format:** PNG or JPG
- **Aspect Ratio:** 9:19.5 (iPhone style)

The iPhone mockup frames will automatically wrap around your screenshots on the landing page.
